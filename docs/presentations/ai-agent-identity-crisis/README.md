# AI Agent Identity Crisis

A 100-slide technical presentation expanding the "AI Agent Identity
Crisis" blog post into a conference / internal-training talk.

## Files

| File | Purpose |
|------|---------|
| `ai-agent-identity-crisis.pptx` | The deck (open in PowerPoint / LibreOffice / Google Slides) |
| `build_deck.py` | Python builder that produces the PPTX |
| `build_charts.py` | matplotlib generators for the 11 embedded chart images |
| `assets/` | Generated chart PNGs (regenerated by `build_charts.py`) |

## Rebuilding

```bash
pip install python-pptx matplotlib numpy
python build_charts.py   # regenerates all chart PNGs
python build_deck.py     # writes ai-agent-identity-crisis.pptx
```

## Deck outline (100 slides, ~60-75 min with Q&A)

1. Section 1 — The agent landscape (slides 6-15)
2. Section 2 — What agent identity means (slides 16-24)
3. Section 3 — Why OAuth breaks (slides 25-33)
4. Section 4 — Prompt injection is an identity problem (slides 34-42)
5. Section 5 — The five-property delegation chain (slides 43-50)
6. Section 6 — The Quidnug answer (slides 51-63)
7. Section 7 — Attack analysis (slides 64-75)
8. Section 8 — Worked example (slides 76-87)
9. Section 9 — Economics and adoption (slides 88-94)
10. Section 10 — Honest tradeoffs (slides 95-100)

Every slide has full bulleted speaker notes attached.

## Editing

Treat the `.pptx` as the authoritative source for small text edits
made through PowerPoint. For layout changes, section reordering, or
new charts, edit the Python builders and re-run.

Speaker notes travel with the PPTX and are visible in PowerPoint's
presenter view.
