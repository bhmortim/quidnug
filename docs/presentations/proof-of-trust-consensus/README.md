# Proof of Trust vs Consensus Mechanisms

90-slide technical presentation expanding the
[Proof of Trust vs Nakamoto, BFT, FBA, and DAG Consensus blog post](../../../blogs/2026-04-20-proof-of-trust-vs-consensus-mechanisms.md)
into talk format.

## Files

| File | Purpose |
|------|---------|
| `proof-of-trust-consensus.pptx` | The deck (open in PowerPoint, OpenOffice, Google Slides) |
| `build_deck.py` | Python builder that produces the PPTX |
| `build_charts.py` | matplotlib generators for the 12 embedded chart images |
| `assets/` | Generated chart PNGs (regenerated by `build_charts.py`) |

## Rebuilding

```bash
pip install python-pptx matplotlib numpy
python build_charts.py   # regenerates all chart PNGs
python build_deck.py     # writes proof-of-trust-consensus.pptx
```

## Outline (90 slides, ~60 min with Q&A)

1. Opening (slides 1-5): hook, agenda, takeaways
2. Section 1: The consensus problem (6-14)
3. Section 2: Tour of existing mechanisms (15-25)
4. Section 3: Their shortcomings (26-33)
5. Section 4: Proof of Trust (34-41)
6. Section 5: The math of relational trust (42-50)
7. Section 6: Security properties (51-57)
8. Section 7: Real-world use cases (58-66)
9. Section 8: Benchmarks and tradeoffs (67-77)
10. Section 9: Decision framework + closing (78-90)

Every slide carries full bulleted speaker notes (Presenter View).
