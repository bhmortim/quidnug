# Relativistic Ratings: The End of Review Spam

90-slide technical presentation expanding the
[Relativistic Ratings blog post](../../../blogs/2026-04-20-relativistic-ratings-end-of-review-spam.md)
into talk format.

## Files

| File | Purpose |
|------|---------|
| `relativistic-ratings.pptx` | The deck (open in PowerPoint, OpenOffice, Google Slides) |
| `build_deck.py` | Python builder that produces the PPTX |
| `build_charts.py` | matplotlib generators for the 12 embedded chart images |
| `assets/` | Generated chart PNGs (regenerated by `build_charts.py`) |

## Rebuilding

```bash
pip install python-pptx matplotlib numpy
python build_charts.py   # regenerates all chart PNGs
python build_deck.py     # writes relativistic-ratings.pptx
```

## Outline (90 slides, ~60-70 min with Q&A)

7. Section 6: The four-factor math (51-60)
8. Section 7: Anti-spam properties (61-67)

Every slide carries full bulleted speaker notes (Presenter View).
